# Data Professional Survey Dashboard (Power BI)

This project analyzes survey data of data professionals to understand salary trends, job roles, skills, and satisfaction levels.

## Key Insights
- Average salary by job role
- Programming language preferences
- Work-life balance & salary happiness
- Difficulty of entering data roles
- Country-wise survey distribution

## Tools Used
Power BI, Excel, Data Cleaning, Data Visualization
